# Welcome to Codr's Storybook

All of our react components can be found here. On the left, you will find a
list of components, each containing a documentation page and several "stories."

A story is a predefined set of arguments that are passed onto the component to
display it in various styles. Check out the written stories in `src/stories/*.stories.tsx`

## File Structure

<div className="prose prose-xl">
```
/                               // project root
  components/                   // components directory
    ComponentName/              // component
      types/                    // component types dir
        ComponentNameProps.ts   // props for component
      index.tsx                 // single point of entry for export
  stories/                      // stories
  styles/                       // global styles
  types/                        // global types
  utils/                        // utilities
```
</div>
