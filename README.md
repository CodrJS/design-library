# @codrjs/design-library

Codr's react design library.

## Development

### Prerequisites

```bash
# Install yarn if you don't have it already.
npm install -g yarn

# Install dependencies
yarn install
```

### Storybook

This repository uses a tool called Storybook to visualze and manually test components.

```bash
# run yarn storybook to get started.
yarn storybook
```
